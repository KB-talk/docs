                              # 一、特征值与特征向量的定义

                              Aα=λαA \alpha = \lambda \alphaAα=λα(其中为AAA矩阵、λ\lambdaλ为特征值、α\alphaα为特征值对应的特征向量)

                              如何理解特征值与特征向量

                              我们应该知道，从几何的角度上理解矩阵是由多个向量组成的。矩阵的相乘的本质是对矩阵的线性变换，单独的一个矩阵即可理解为对单位矩阵进行了多次的线性变换，而向量是具有方向属性的，矩阵即是沿着特征向量的方向放缩了特征值的倍数。

                              # 四、对角化

                              通过上面方法求出的特征值与特征向量,将特征值组成对角矩阵⋀\bigwedge⋀，对应的特征向量组成矩阵PPP,使得矩阵相似于对角矩阵P−1AP=⋀P^{-1}AP= \bigwedgeP−1AP=⋀

                              存在矩阵PPP可使得P−1AP=⋀P^{-1}AP= \bigwedgeP−1AP=⋀，则矩阵AAA可以对角化

                              提示

                              对角化是通过PPP这个基变换矩阵，将现有坐标映射到另一个坐标系，在新的坐标系下的坐标会变得简单，即对角矩阵。

                              # 五、合同

                              矩阵PPP中的向量不一定是相互正交的，对PPP进行正交化得到矩阵QQQ，使得QTAQ=⋀Q^TAQ=\bigwedgeQTAQ=⋀，两个矩阵A和B是合同的，当且仅当存在一个可逆矩阵QQQ，使得QTAQ=⋀Q^TAQ=\bigwedgeQTAQ=⋀。

                              合同判断：两个实对称矩阵合同的充要条件是它们的正负惯性指数相同

                              # 六、相似

                              相似的含义

                              P−1AP=BP^{-1}AP= BP−1AP=B

                              相似的性质

                              • 特征值相同（特征向量不一定相同）

                              • 主对角线元素之和相同

                              • r(A)=r(B)r(A)=r(B)r(A)=r(B)

                              • ∣A∣=∣B∣|A|=|B|∣A∣=∣B∣

                              • A∼B⇔A+kE∼B+kEA \sim B \Leftrightarrow A+kE \sim B+kEA∼B⇔A+kE∼B+kE

                              • A∼B⇔An∼BnA \sim B \Leftrightarrow A^n \sim B^nA∼B⇔An∼Bn

                              可利用性质判断两个矩阵是否相似

                              相似的判断

                              若AAA与BBB的特征值相同且都可对角化 ⟹\Longrightarrow⟹ AAA与BBB相似

                              # 七、相似对角化

                              相似对角化是可以与对角矩阵相似

                              可相似对角化的判断方法

                              充分条件：

                              • 实对称矩阵一定可以相似对角化

                              • 没有相同特征值

                              充要条件：

                              • nnn重特征值对应nnn个特征向量

                              • 若AAA的特征值λ\lambdaλ是kkk重跟，则n−r(λE−A)n-r(\lambda E-A)n−r(λE−A)需等于kkk

                              # 八、实对称矩阵

                              # 定义

                              AT=AA^T=AAT=A

                              # 性质

                              可相似对角化

                              有nnn个线性无关的特征向量

                              kkk重根有kkk个线性无关的特征向量

                              不同特征值对应的特征向量相互正交

                              可用正交矩阵QQQ对角化QTAQ=⋀Q^TAQ=\bigwedgeQTAQ=⋀

                              特征值是实数

                              小结

                              本结的核心是对对角化的理解，对角化主要解决的是对复杂的矩阵AAA通过转换基坐标PPP的方法简化矩阵，特征值和特征向量是用于求解PPP和⋀\bigwedge⋀的工具。相似的含义是矩阵AAA经过矩阵PPP可以转换为矩阵BBB，若矩阵AAA和矩阵BBB都可以对角化同一个对角矩阵，则矩阵AAA和矩阵BBB可相似对角化。合同可以看作更加特殊的对角化。
