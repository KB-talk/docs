## 一、二次形的矩阵表示

平方项为主对角线，其余的按照对应位置平分。

## 二、二次型的标准形与规范形

### 二次型的秩

二次形的秩即矩阵的秩

### 二次型的标准型

二次型化为标准型方法

1. **配方法**

**情况一**：含平方项

通过加减相同的平方项，逐项凑平方项

**情况二**：不含平方项

$\begin{cases}x_1=y_1-y_2 \\x_2=y_1+y_2 \\x_3=y_3\end{cases}$

3. **正交变换法$x=Qy$**

**step1**:求出矩阵的特征值

**step2**:求出特征值对应的特征向量

**step3**:对特征向量进行单位正交化，组成矩阵$Q$

**step4**:求得二次型的标准型$y=\lambda _1y_1^2+\lambda _2y_2^2+\lambda _3y_3^2$

### 正负惯性指数

1. 正惯性指数：正特征值的个数

2. 负惯性指数：负特征值的个数

3. 惯性定理：同一个二次型的正负惯性指数相同。

4. 惯性指数与空间曲面的对应关系

|   惯性指数   |  空间曲面  |
| :----------: | :--------: |
|     三正     |    椭球    |
|   二正一负   | 单叶双曲面 |
|   一正二负   | 双叶双曲面 |
|   二正一零   |  椭圆柱面  |
| 一正一负一零 |   双柱面   |

### 二次型的规范性

平方项的系数只有$-1,0,1$的标准型

## 四、正定二次型

二次型的标准型的系数全为正的二次型

正定二次型矩阵是对称矩阵

### 必要条件

矩阵的所有元素都为正

### 充要条件

特征值都为正
