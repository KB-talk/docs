## 一、线性方程组的解

### 齐次

|        |     唯一解     |   无穷解   |
| :----: | :-------------: | :---------: |
|   秩   |   $r(A)=n$   | $r(A)<n$ |
| 行列式 | $\|A\| \ne 0$ | $\|A\|=0$ |

### 非齐次

|        |      唯一解      |      无穷解      |      无解      |
| :----: | :---------------: | :---------------: | :-------------: |
|   秩   | $r(A)=r(A,b)=n$ | $r(A)=r(A,b)<n$ | $r(A)<r(A,b)$ |
| 行列式 |  $\|A\| \ne 0$  |    $\|A\|=0$    |   $\|A\|=0$   |

## 二、抽象方程组的解
### 解的结构
1. 若$\alpha _1,\alpha _2$是$Ax=0$的解，则$k_1\alpha _1+k_2\alpha _2$也是$Ax=0$的解

2. 若$\alpha _1,\alpha _2$是$Ax=b$的解，则$k_1\alpha _1+k_2\alpha _2$也是$Ax=(k_1+k_2)b$的解；若$k_1+k_2=0$，$k_1\alpha _1+k_2\alpha _2$也是$Ax=0$的解；若$k_1+k_2=1$，$k_1\alpha _1+k_2\alpha _2$也是$Ax=b$的解；

3. $Ax=b$的解是基础解系的线性组合

4. $Ax=b$的通解=齐次通解+一个非齐次特解

### 基础解系
定义：解集的极大线性无关组。

基础解系所含向量的个数：$n-r(A)$。
