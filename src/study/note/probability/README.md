## 概述

概率论与数理统计的核心其实是概率问题。在我们高中就学过概率，概率简单的说就是某件事发生的可能性。在概率论与数理统计这门学科对此事情进行了严谨科学的研究。

首先，在我们的日常的生活中，经常提到明天可能下雨。在这句话中，下雨就可以看作事件，有多大的肯能性就是概率。

> 现实中，一定发生的现象确定现象，有可能发生的是随机现象。在这里提到的是现象而不是事件。那随机事件又是什么呢？

说到随机事件，首先就要理解随机试验。随机试验就是比较科学的说法了，随机试验可以理解我们做了多次的试验，我们可以知道每次的试验结果，但是下次的结果还是未知的。理解了什么是随机试验，就来看看什么是样本点和样本空间吧。对于一个随机试验，每次试验都会出现结果，这个结果就称为样本点，而所有可能出现的结果就是样本空间。

> 在数学中，事件是存在随机试验中的，事件就是试验结果。由单一结果组成的样本点可称为基本事件。事件是很宽泛的，简单来说事件就是描叙试验结果的。

上面说了一大堆，说的都是如何区分现象、随机试验、样本、事件的概念。那么下面就说说什么是频率和概率吧。

有了随机事件，我们更希望知道事件有多大可能发生。那么这个可能性就是概率，那么概率有多大应该怎么知道呢？前文说到随机事件是因为做了多次试验，做了试验就有了试验的结果，我们就可以计算发生的次数与试验的次数的关系来的到可能性，这个关系就是事件发生的频率。然而，当我门进行了大量的试验发现，这个频率稳定在了一个常数，有了这个常数，我们就可以认为有多大的概率这个事件会发生。

不同事件之间会存在有一定的关系，具体会有哪些关系在这里不细说。

::: info 小结

说了这么多，对上面的内容做一个总结。核心内容就是理解事件与概率。在数学中，为了研究某一现象发生的可能性有多大，于是进行大量的随机事件，每次试验结果称为样本点，也可称为事件。对试验的结果进行统计，可以知道事件发生的概率。

:::

从这里开始，与高中的概率开始有了很大的区别了。提出了一个新的概念————随机变量。如何理解随机变量与随机事件的关系呢。在上面我们提到的随机事件大部分都是用语言描述的，并不能使用数学的方法来处理，于是提出了随机变量的概念。随机变量的提出，使概率论与数理统计的研究提升了新的高度，我们可以使用学过的微积分来处理概率问题。

随机变量如此的重要，那么我们应该如何理解他呢？随机变量的本质是用数字来表示随机事件，比如学生的身高，有165、170、168、175等等。我们就可以使用X=165来描述学生身高为165这个之间。使用$P{X=165}=0.3$来表示学生身高为165的概率为0.3。使用这个方法就可以很简单表示出事件与概率的关系。

上面的描述看起来是不是很简单，无论是表示事件还是概率都没什么问题，因为上面的例子是属于离散型的，那么对于连续形的表示起来就没那么方便了，对于连续形我们只能$P{X \le x}$这种方式来表示了，每次使用这种写法是很麻烦的，于是提出了分布函数这个概念，他的含义是随机变量小于某一值的概率为多少。对分布函数进行求导还可以得到概率密度。

> 使用分布函数表示概率有个好处就是，无论是连续还是离散都可以进行表示。对于连续形的随机变量，可以对分布函数进行求导从而得到概率密度

上文，提到的随机变量的概念，无论是针对离散还是连续形都是适合的。经过长期研究，人们找到了几种常见的随机变量及其分布。考试主要考的内容主要围绕着常见概率分布于概率密度。

知道了什么，是随机变量、概率分布、概率密度。接下来，就要知道什么是随机变量的数字特征。数字特征主要包含随机变量的数学期望、方差、协方差、标准差还有矩。其研究内容主要是对不同的分布函数进行相应的计算，具体计算方法不在这里介绍。

::: info 提示
以上内容默认为以为随机变量，对于二维随机变量，对于定义的理解，无疑是多了一个随机变量，没有太大的区别，具体的计算方法在对应的章节中进行说明。
:::

以上提到的都是由随机变量组成的简单的随机变量分布，那么由简单随机变量能否组成新的随机变量呢？组成新的随机变量还会服从什么分布函数？这是我门接下来讨论的内容。到此，我们又提出了随机样本的概念，什么是随机样本呢，回想上文提到的随机事件，随机样本简单理解就是随机变量的可能的取值中的一个，我们对随机样本进行观察得到的结果就是样本值。

统计量又是什么呢？统计量本质是以随机变量为参数的方程，是对样本的一种处理，有几种常见的处理方法有样本均值、样本方差、样本原点矩、k阶中心矩。
经过统计量处理后，还是需要知道分布函数的，这个分布函数称为抽样分布。几种常见的分布只是当随机变量满足一种分布，经过统计量处理后服从新的分布。

到此，以上都是在已知概率分布进行研究的，在题目都会告诉随机变量服从什么分布或者已知分布函数。接下来讨论的是已知随机变量服从什么概率分布，但是是缺少参数的，研究的目的就是尽可能的求得未知的参数。点估计、极大似然估计都是从总体中抽取一定的简单随机样本进行对总体进行估计。
